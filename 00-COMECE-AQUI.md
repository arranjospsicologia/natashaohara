## 💡 Dicas Importantes

### Para Usar o Site:
1. **Não precisa instalar nada** - só fazer upload
2. **Funciona em qualquer dispositivo** - testado em mobile/desktop
3. **Zero custo** - GitHub Pages é gratuito
4. **Fácil atualizar** - edita o arquivo e faz commit

### Para Atualizar Conteúdo:
- **Preços:** Edite `index.html` na seção `id="servicos"`
- **Horários:** Edite `index.html` na seção `id="contato"`
- **Textos:** Edite diretamente no HTML
- **Cores:** Edite `css/styles.css` nas variáveis CSS

### Para Adicionar Fotos:
1. Crie pasta `images/` no repositório
2. Faça upload das fotos
3. Adicione no HTML: `<img src="images/foto.jpg" alt="Descrição">`

## 🆘 Problemas Comuns

### Site não aparece após ativar GitHub Pages
**Solução:** Aguarde 5-10 minutos e limpe o cache do navegador

### Menu mobile não abre
**Solução:** Verifique se o arquivo `js/script.js` está carregando

### Widget Instagram não funciona
**Solução:** Normal! Precisa configurar seguindo o QUICKSTART.md

### Cores diferentes do manual
**Solução:** Não deve acontecer, mas verifique `css/styles.css` nas variáveis `:root`
